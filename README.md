**Time-Off Dashboard (Frontend)**

A React + TypeScript dashboard for managing employee time-off requests.

** Features**

* View leave balance
* Submit time-off requests
* Sync data with backend
* Dark / Light mode

** Tech Stack**

* React (TypeScript)
* Vite
* Ant Design
* Axios

** Setup Guide**

Clone Repository

git clone https://github.com/Daniyal-Aslam/timeoffservice-ui.git
cd timeoffservice-ui

**Install Dependencies**

npm install

** Setup Environment**

Create .env file:

VITE_API_BASE_URL=http://localhost:3000


**Start App**

npm run dev


**Open in Browser**

http://localhost:5174


**Requirement**

Backend must be running at:

http://localhost:3000
